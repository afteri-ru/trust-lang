# types — Система типов

Определение всех типов языка, правила совместимости типов, регистрация user-defined типов (enum, struct).

## Ответственность

- Описание примитивных типов (`TypeKind`, `TypeInfo`)
- Правила совместимости и неявных преобразований
- Парсинг строковых представлений типов

## Зависимости

- `diag` — для `SourceLoc` в ошибках
