# jit — JIT-компиляция
- 
Компиляция AST в машинный код через LLVM ORC JIT, symbol resolution, управление памятью выполнения.

## Ответственность

- Компиляция AST в выполняемый код в памяти
- Разрешение символов (symbol resolution)
- Управление памятью выполнения

## Зависимости

- `types` — система типов
- `variable` — type-erased контейнеры
- `runtime` — среда выполнения
- `optimizer` — AST-оптимизации
- `gencpp` — AST-структуры
