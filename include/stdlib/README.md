# stdlib — Стандартная библиотека
и
Встроенные функции: `print`, `len`, `assert`, математика, ввод/вывод, коллекции.

## Ответственность

- Функции вывода (`print`, `println`)
- Функции работы с коллекциями (`len`, `append`, `slice`)
- Математические операции
- Утилиты (`assert`, `panic`)

## Зависимости

- `types` — система типов
- `runtime` — среда выполнения
