# repl — Интерактивная оболочка

Цикл read-eval-print, история команд, автодополнение, инкрементальная компиляция через JIT.

## Ответственность

- Интерактивный цикл чтения-выполнения-печати
- История команд и автодополнение
- Инкрементальная компиляция через JIT

## Зависимости

- `driver` — конвейер компиляции
- `jit` — JIT-компиляция
- `stdlib` — стандартная библиотека
