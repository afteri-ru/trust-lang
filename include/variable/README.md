# variable — Универсальный type-erased контейнер

Value-обёртка для хранения значений любого `TypeKind` без потери типа. Используется в runtime, JIT, REPL для динамического выполнения.

## Ответственность

- Type-erased хранение значений любого типа языка
- Конвертация между типами при необходимости
- Поддержка boxing/unboxing для dynamic dispatch

## Зависимости

- `types` — система типов
- `runtime` — среда выполнения
