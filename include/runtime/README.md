# runtime — Среда выполнения и стандартная библиотека

Среда выполнения: 
- Обработка исключений (try-all/try+ механизмы)
- Type-erased контейнер значений любого `TypeKind` (boxing/unboxing, динамический dispatch)
- Инициализация стандартной библиотеки при запуске и встроенные функции

## Зависимости

- `types` — система типов
