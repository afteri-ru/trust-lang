# runtime — Среда выполнения

Обработка исключений (try-all/try+), подсчёт ссылок, управление памятью, инициализация stdlib при запуске.

## Ответственность

- Обработка исключений (try-all/try+ механизмы)
- Подсчёт ссылок (reference counting)
- Управление памятью
- Инициализация стандартной библиотеки при запуске

## Зависимости

- `types` — система типов
